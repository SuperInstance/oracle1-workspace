# Skill Template

Copy this folder to create a new skill. Replace ALL {placeholders}.

## Quick Deploy
```bash
# 1. Copy template
cp -r SKILLS/templates/{blank-skill} SKILLS/active/{your-skill-name}

# 2. Edit SKILL.md
# 3. Add src/ and tests/
# 4. Run tests
# 5. Commit
```

## Template Files
- SKILL.md — this file, filled in
- src/ — your implementation
- tests/ — your verification
- MANIFEST.json — auto-generated state tracker
