**6 real gaps identified:**
1. No A2A compliance — can't talk to CrewAI/AutoGen/etc agents

3. Zero observability — no tracing, evaluation, cost tracking

5. No semantic search — keyword only, no embeddings
6. No Agent Cards — no standard way to advertise fleet capabilities

1. Expose PLATO as MCP server (~200 lines, any MCP agent can query our knowledge graph)
